# 为 VSCode 设置字体

选择一款花里胡哨的字体可能也还不错？

表面上是介绍 Fira Code 字体，实际上是安利一个好用的站点 slant (opens new window)。

# 步骤

# 1. 下载字体

首先需要前往 slant (opens new window) 寻找感兴趣的字体，搜索 font，得到搜索结果 (opens new window)，再找到 Fira Code 的下载地址 (opens new window)。

# 2. 安装字体

提供该安装包的 Github 有详细的安装说明。

在 manjaro 下依次选择 Font Management > Personal Fonts > Install from File，最后选择解压的 .ttf 文件即可。

# 3. VSCode 中启用

打开 Settings，输入 Font Family，在已有的一系列字体前面新增 Fira Code ，如果希望一些符号具有连体字(Font Ligatures)效果，同样需要在设置中启用它。
